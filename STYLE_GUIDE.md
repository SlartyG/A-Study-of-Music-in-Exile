# Style Guide — «Музыка после отъезда»
### Концепция «Досье»

*Интерактивный веб-лонгрид. Версия 1.0 — июнь 2026*

---

## Суть концепции

Визуальный язык — архив, который рассекречивают прямо сейчас. Читатель не просто листает статью, он листает дело. Каждый артист — карточка в картотеке. Каждый вывод — строчка в отчёте. Эстетика намеренно отсылает к государственным документам: это не случайно, тема сама диктует этот язык — люди в исследовании получили официальные статусы иностранных агентов и экстремистов, их имена фигурируют в реестрах.

Никакого пафоса. Никакой драмы. Холодная точность — и от этого ещё холоднее.

## Цветовая палитра

### Базовые цвета

| Название | HEX | Применение |
|---|---|---|
| `--black` | `#0D0D0D` | Фон страницы |
| `--paper` | `#F0EBE0` | Фон «документов», карточек артистов |
| `--ink` | `#1A1A1A` | Основной текст на светлом фоне |
| `--ash` | `#3A3A3A` | Вторичный текст, подписи |
| `--fog` | `#8A8A8A` | Даты, метки, деактивированные элементы |

### Акценты

| Название | HEX | Применение |
|---|---|---|
| `--stamp-red` | `#C1272D` | Штампы ОПС, экстремальные значения на графиках, метки «После» |
| `--stamp-amber` | `#D4A017` | Выделитель-маркер, hover-состояния, метки «До» |
| `--typewriter` | `#4A3728` | Моноширинный текст, машинописные вставки |

### Правила использования цвета

- Красный — только для содержательных акцентов: статус ОПС, статистически значимый сдвиг, отрицательная дельта. Не декоративно.
- Янтарь — интерактивные элементы и выделения. Буквально как след маркера.
- На тёмном фоне (заголовочные блоки, переходы) — белый текст `#F5F5F5`, акценты те же.
- Большинство страницы — бежевая «бумага». Тёмный фон только у hero-секции и разделительных экранов.

## Типографика

### Шрифтовая система

**Заголовки** — [Playfair Display](https://fonts.google.com/specimen/Playfair+Display), serif  
Газетный характер, высокий контраст штрихов. Подходит для H1–H3.  
Альтернатива: Cormorant Garamond (более вытянутый, аристократичнее).

**Основной текст** — [Lora](https://fonts.google.com/specimen/Lora), serif  
Читабельный, умеренный. Хорошо работает на светлом фоне в длинных абзацах.  
Альтернатива: PT Serif (русскоязычная поддержка лучше).

**Моноширинный текст** — [JetBrains Mono](https://www.jetbrains.com/legalnotice/fonts/) или Courier New  
Для цифр, кодов, штампов, машинописных вставок, дат в карточках артистов.  
Обязательно — слегка пожелтевший цвет `--typewriter`.

**Интерфейсные элементы** — [Inter](https://fonts.google.com/specimen/Inter), sans-serif  
Навигация, кнопки, легенды к графикам, подписи осей.

### Размеры (desktop, 1440px)

```
H1 (hero)       — 72–96px, Playfair, weight 700, letter-spacing: -0.02em
H2 (раздел)     — 48px,    Playfair, weight 700
H3 (подраздел)  — 32px,    Playfair, weight 600, italic
H4 (карточка)   — 24px,    Playfair, weight 600
Body            — 18px,    Lora, line-height: 1.75
Caption         — 14px,    Inter, weight 400, color: --fog
Mono label      — 13px,    JetBrains Mono, weight 500
Stamp           — 20–40px, JetBrains Mono, UPPERCASE, letter-spacing: 0.15em
```

### Типографические детали

- Кавычки — «елочки» для русского текста.
- Тире — длинное (—), не дефис.
- Числа в тексте — моноширинным шрифтом, чтобы выделялись.
- «Гриф» (ИНОСТРАННЫЙ АГЕНТ, ЭКСТРЕМИСТ) — капслок, красный, моно, opacity: 0.85.

## Текстуры и эффекты

### Зернистость

На всех фоновых блоках — тонкий шум. Реализация через SVG-фильтр или CSS:

```css
.noisy {
  position: relative;
}
.noisy::after {
  content: '';
  position: absolute;
  inset: 0;
  background-image: url("noise.svg");
  opacity: 0.04;
  pointer-events: none;
}
```

На тёмном фоне — opacity: 0.06. На светлом — 0.03.

### Линии сканера

Горизонтальные полосы с очень низким opacity (~0.015) поверх hero-секции — имитация засветки от сканера. Статичные, не анимированные.

### Мятая бумага

Карточки артистов и блоки-документы — лёгкая тень `box-shadow` с тёплым оттенком:
```css
box-shadow: 2px 3px 12px rgba(60, 40, 20, 0.18);
```

### Штамп

Элемент `.stamp` — повёрнут на -15° или +12°, красный border, моноширинный текст. Используется для статусов ОПС. Не злоупотреблять — максимум один на карточку.

### Общая структура

```

│  HERO (тёмный фон, заголовок + лид)          │
├─────────────────────────────────────────────┤
│  ВВЕДЕНИЕ (бумага, 2 колонки)               │
├─────────────────────────────────────────────┤
│  МЕТОДОЛОГИЯ (accordion, 1 колонка узкая)   │
├─────────────────────────────────────────────┤
│  АРТИСТЫ — горизонтальный скролл картотеки  │
├─────────────────────────────────────────────┤
│  РЕЗУЛЬТАТЫ — scrollytelling секции         │
│   ├─ Тональность: violin + текст            │
│   ├─ ОПС vs без ОПС                        │
│   └─ Темы: LDA heatmap                     │
├─────────────────────────────────────────────┤
│  ВЫВОДЫ (тёмный фон, крупный текст)         │

```

### Сетка

- Максимальная ширина контента: **860px** для текста, **1200px** для визуализаций.
- Отступы страницы: 48px (desktop), 24px (tablet), 16px (mobile).
- Базовая единица: 8px.

### Боковая навигация

Фиксированная слева (только desktop), стилизована под вертикальную опись:

```
РАЗДЕЛ 1   ВВЕДЕНИЕ
РАЗДЕЛ 2   КОРПУС
РАЗДЕЛ 3   МЕТОДОЛОГИЯ
РАЗДЕЛ 4   АРТИСТЫ
РАЗДЕЛ 5   РЕЗУЛЬТАТЫ
РАЗДЕЛ 6   ВЫВОДЫ
```

Моноширинный шрифт, маленький, серый. Активный раздел — янтарный.

## Инфографика и визуализации

### Общие правила для всех графиков

1. **Фон** — `#F0EBE0` (та же «бумага»). Никаких белых прямоугольников на белом фоне.
2. **Сетка** — тонкие горизонтальные линии `#D8D0C4`, без вертикальных.
3. **Шрифты на графиках** — Inter (метки осей), JetBrains Mono (числовые значения).
4. **Цвет данных:**
   - Период «До» — `#4A7FA5` (приглушённый синий, архивный)
   - Период «После» — `#C1272D` (красный)
   - Нейтральный / весь корпус — `#7A6E5E`
5. **Линии** — weight 1.5px, без маркеров (кроме scatter).
6. **Никаких 3D**, теней на данных, градиентных заливок баров.

### Violin-диаграммы (final_2)

Violin заливается полупрозрачно (opacity 0.45), контур — 1.5px. До/После рядом на одной оси. Внутри — точки медианы и IQR, без боксов (слишком тяжело визуально).

Подпись: «Каждая точка — один трек» — мелким Inter под осью.

### Scatter-плоты (final_5)

Каждая точка — артист. Размер точки пропорционален числу треков. Точки с ОПС — форма квадрат, без ОПС — круг. Hover показывает карточку: имя + два значения + статус.

### Таймлайн (final_6)

Горизонтальная линия времени. Точка релокации каждого артиста — вертикальная засечка со штампом-датой под ней (моно, мелко). Тональность до и после — плавная кривая, два цвета.

### Heatmap тем (final_7 / LDA)

Ячейки — тёплая монохромная шкала от `#F0EBE0` до `#3A2010`. Метки тем — моноширинным, курсивом. Никакого rainbow. Hover на ячейке — тултип с топ-словами темы.

### Таблица-сводка (final_1)

Стилизована под машинописный документ. Заголовки столбцов — капслок, Inter. p-value < 0.05 — выделено янтарём. Строки с ОПС — лёгкий красный tint на фоне строки.

### Delta-диаграмма по артистам (final_3)

Горизонтальный dot-plot. Нулевая линия — пунктир. Положительная дельта — синий, отрицательная — красный. Значимые артисты (p < 0.05) — точка крупнее + имя жирнее.

## Карточки артистов («Картотека»)

Горизонтальный ряд карточек, скролл мышью/свайп. Или сетка 3×N на десктопе.

### Структура карточки

```
┌──────────────────────────────┐
│ [фото / аватар в ч/б]        │
│                              │
│ ПСЕВДОНИМ                    │  ← Playfair, 20px
│ Настоящее имя                │  ← Lora italic, 14px, --fog
│                              │
│ Релоцировался: МАР 2022      │  ← Mono, 12px
│ Страна: Грузия               │  ← Mono, 12px
│                              │
│ [ШТАМП: ИНОСТРАННЫЙ АГЕНТ]   │  ← если есть ОПС
│                              │
│ Треков до: 47 / После: 31    │  ← Inter, 12px
│ Δ тональность: −0.008 n.s.   │  ← Mono, цвет по знаку
└──────────────────────────────┘
```

Фотографии — чёрно-белые, с зерном. Если фото нет — серый прямоугольник с инициалами.

Клик на карточку — открывает боковую панель с подробной визуализацией по артисту.

## Анимации и взаимодействие

### Принципы

- Анимации — только при скролле (scroll-triggered), не при загрузке.
- Ничего не мигает и не прыгает само по себе.
- Все длительности — 300–600ms, easing: `ease-out`.

### Конкретные сценарии

**Hero-секция**  
Заголовок появляется построчно с задержкой — как будто печатается. Cursor-мигалка в конце, исчезает через 2 секунды.

**Секции с данными (scrollytelling)**  
Пока пользователь не доскроллил до графика — тот размыт (blur: 6px, opacity: 0.3). При входе в viewport — проявляется за 500ms.

**Карточки артистов**  
При hover — лёгкий подъём (translateY: -4px), тень усиливается. Эффект «достал из стопки».

**Числа в тексте**  
При входе в viewport — анимированный счётчик от 0 до нужного значения. Только для ключевых цифр: 21 артист, 1598 треков.

**Штампы ОПС**  
Появляются с лёгким «хлопком» — scale от 1.3 до 1.0 за 200ms.

**Проявление документов**  
Блоки-документы slide-in снизу (translateY: 30px → 0) при скролле.

### Чего не делать

- Никакого parallax на тексте — нечитабельно.
- Никаких автоплеящихся видео или звуков.
- Никаких pop-up оверлеев поверх текста.

## Мобильная версия

Приоритет — читаемость. Визуализации адаптируются, а не уменьшаются.

| Элемент | Desktop | Mobile |
|---|---|---|
| Боковая навигация | Фиксированная слева | Hamburger сверху |
| Картотека | Горизонтальный скролл | Вертикальный стек |
| Violin-диаграммы | 2 рядом | По одной |
| Scatter | Полный | Полный (zoom) |
| Карточки артистов | 3 в ряд | 1 в ряд |
| Таймлайн | Горизонтальный | Вертикальный |

Шрифты на mobile: H1 — 40px, body — 16px, line-height: 1.8 (длиннее строки — нужнее интерлиньяж).

## Структура лонгрида

### 1. Hero

Тёмный фон. Заголовок «Музыка после отъезда» — крупно, Playfair. Подзаголовок — Lora italic, белый, smaller. Ниже — три цифры в ряд как «параметры дела»:

```
21 АРТИСТ     1 598 ТРЕКОВ     2022–2023
```

Кнопка-стрелка вниз. Никакого вступительного текста здесь — его читают дальше.

### 2. Что произошло

Короткий раздел — 3–4 абзаца. Пятая волна эмиграции, масштаб, кто такие эти люди. Без патетики.

### 3. Как мы считали

Accordion с тремя методами. По умолчанию закрыт — читатель сам решает, нужна ли методология. Внутри — краткие объяснения без формул.

### 4. Артисты

Картотека. Над ней — фильтры: «Все / С ОПС / Без ОПС / По стране».

### 5. Тональность: что показывают данные

Scrollytelling. Слева — текст (тезис), справа — график проявляется по мере чтения.

Последовательность:
1. Violin всего корпуса → «тональность отрицательная в обоих периодах» (жанровая норма)
2. Таблица-сводка → «в среднем сдвига тональности нет» (p > 0,05 на балансированной выборке)
3. Delta по артистам → «но у конкретных людей — разброс, у части значимо»
4. Темы (LDA) → «тематический состав меняется устойчивее, чем „настроение“»
5. Таймлайн по годам (опционально) → контекст, с календарным делением до/с 2022

Подробная карта PNG → интерактив: [LONGREAD_VISUALS.md](LONGREAD_VISUALS.md).

### 6. Темы: о чём стали петь иначе

LDA-heatmap + текстовые пояснения. Какие темы выросли, какие упали.

### 7. Выводы

Тёмный фон, крупный текст. Шесть выводов — коротко, без воды. Каждый — отдельный «абзац-тезис».

### 8. Оговорки

Серый блок внизу. Мелко. Методологические ограничения, правовая ремарка об ОПС.

## Технический стек (рекомендация)

| Задача | Инструмент |
|---|---|
| Фреймворк | Next.js (SSG) или SvelteKit |
| Визуализации | Observable Plot + D3 для кастомных |
| Scrollytelling | Svelte-scroller или Scrollama (JS) |
| Анимации | Framer Motion (React) / CSS transitions |
| Шрифты | Google Fonts (Playfair, Lora, Inter) + JetBrains Mono |
| Деплой | Vercel или GitHub Pages |

Альтернатива без программирования: **Readymag** или **Webflow** с ручными SVG-иллюстрациями вместо D3.
